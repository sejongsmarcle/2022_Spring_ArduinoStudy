# 느낀점
사실 처음에 강의를 보고 따라했을때 전구에 불이 안켜져서 당황했었다.
아두이노에서 빵판으로 전류를 보내는 원리를 정확히 이해하지 못해서 회로를 잘못 만들었었다.
다행히 몇번의 시도 끝에 불이 켜졌다.
아두이노와 빵판의 연결 원리를 이해한 것 같아서 다행이다.
